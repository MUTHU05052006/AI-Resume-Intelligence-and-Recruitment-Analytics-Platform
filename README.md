# AI Resume Intelligence and Recruitment Analytics Platform

## Overview
AI-powered recruitment management system developed during internship at Beau Roi Pvt Ltd.

The platform automates candidate management, resume screening, AI-based evaluation, recruitment analytics, and report generation through interactive dashboards.

## Features
- Candidate Data Management
- Resume Parsing (PDF & DOCX)
- AI-Based Resume Scoring
- MySQL Database Integration
- QR Code Candidate Entry
- CSV & Excel Synchronization
- PDF Report Generation
- Matplotlib Analytics Dashboard
- Tableau Interactive Dashboard
- Search, Filter and Sorting Operations

## Technologies Used
- Python
- Tkinter
- MySQL
- Flask
- Pandas
- Scikit-learn
- Matplotlib
- Tableau
- ReportLab
- pdfplumber
- python-docx
